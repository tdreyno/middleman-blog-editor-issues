Middleman Blog Editor

Middleman Blog Editor is a paid extension to Middleman which adds WYSIWYG article editing and a blog organization interface.

This extension makes it possible to host a Middleman Blog and give the editor URL to a client so they can add content. Files are modified in-place, so committing those changes to source control or deploying new builds is up to you.

Purchasing a License

The current release, v0.2, costs $6 and can be purchased online. Purchasing will give access to private Rubygems server (http://gems.tdreyno.com), direct gem file download and full source file download.

Usage

After purchasing, you will recieve a username and password for accessing the private Rubygems repository. Add this private source and gem to your Middleman Gemfile

source :rubygems
source "http://username:password@gems.tdreyno.com"

gem "middleman"
gem "middleman-blog"
gem "middleman-blog-editor"

Next, activate the editor in your config.rb

activate :blog_editor

When you start Middleman, you will now have access to the Blog Editor at: http://localhost:4567/editor

If you'd like to put the editor somewhere else, to avoid clashing with the rest of your site, you can use the mount_at option:

activate :blog_editor do |editor|
  editor.mount_at = "/editor"
end
